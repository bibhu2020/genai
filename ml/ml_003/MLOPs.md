# MLOps Tools

    - Code version control (github)

    - Dataset version control (DVC)

    - Experiment Tracking (MLflow)

    - API Development (e.g. API wrapping a model. credit_risk.ipynb --> model.joblib)

    - Deployment to Cloud

    - Calling API

# Deployment

    - Create CI/CD pipeline to the followings:

        - Export your model ipynb file to pklib or joblib file

        - Build FASTAPI using model.joblib + scaler + metadata

        - Create a Container (model files and dependency)

        - Deploy into Kubernetes
